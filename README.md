## 基于Vue+Node的宠物医院管理系统

该项目主要是基于宠物医院的背景下进行开发，拥有后台管理和移动端的应用app。

后台管理的主要功能有用户管理、医院简介和公告管理、医院人员信息管理、宠物档案中心管理、系统中心设置、请假管理、打卡记录、值班安排等等，

客户端app的主要功能有预约和挂号、医院简介、科室介绍、医院设备环境、医生介绍、个人中心、个人中心包括查看我的预约、查看我的挂号、修改密码、设置、退出登录等
